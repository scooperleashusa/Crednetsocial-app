# CrednetSocial - Backend (starter)

This is a minimal TypeScript + Express starter for the CrednetSocial backend.

Quick start (local):
1. cp .env.example .env && edit
2. npm install
3. npm run dev

Build:
- npm run build
- npm start

Notes:
- This starter uses an in-memory user store. Replace with a real DB (Postgres/Mongo) and update controllers/models.
